Class: PhysicsSystem

PhysicsSystem()

The physics system functions differently from other systems, Rather than attaching components, physical properties such as shape, body, mass, etc are definied as attributes. if shape and body are not defined, they default to the geometry of the entity, if there is no geometry, there is no physics defined on the entity. Alternatively, you can also expressly attach a comp-physics attribute for more detailed control.
